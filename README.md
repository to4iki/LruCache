LruCache
=======

[![License][license-image]][license-url]

Least Recently Used.  
build a thread-safe, in-memory LRU cache with a maximum item count of 10.

1. Should perform well as a web image cache, (avg ~1MB)
2. Should be well tested
3. Simplicity of API
4. Overall performance
5. Thread safety
